# Proyecto: Sitio Web con HTML, CSS y JavaScript

Este proyecto consiste en una página web desarrollada con HTML, CSS y JavaScript.  
Está organizada en carpetas para mantener una estructura clara y modular.  
Incluye vistas para registro, inicio de sesión y navegación principal.

## Estructura del proyecto

- `css/` → Hojas de estilo (`styles.css`, `reset.css`, `alternative.css`)
- `fonts/` → Archivos de tipografías
- `ico/` → Iconos utilizados en la interfaz
- `img/` → Imágenes del sitio
- `js/` → Scripts JavaScript
  - `main.js` → Script principal
  - `controllers/` y `models/` → Lógica modular del sitio
- `views/` → Páginas HTML
  - `home.html`, `landing.html`, `login.html`, `register.html`

## Cómo visualizar el proyecto

1. Abrir el archivo `views/landing.html` en un navegador web.
2. Verificar que las rutas relativas funcionen correctamente dentro de la carpeta `public`.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
